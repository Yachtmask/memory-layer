# Memory Layer

What survives when the network forgets?

Memory Layer is an interactive experiment exploring information persistence across unreliable networks.

Users submit a message.

The message fragments.

Fragments travel.

Some disappear.

The system reconstructs what remains.

---

## Concept

Data moves.

Meaning survives.

Inspired by ideas from modern network coding and distributed propagation systems.

## Stack

- Next.js
- TypeScript
- Tailwind
- Framer Motion

## Run

npm install

npm run dev
